### Requirements

- *nix os
- `node ~6.6.0`
- `npm ~3.10.8`


### Install

```
npm run setup
```

### Run in dev

```
npm run start:api
npm run start:webpack
```

navigate to http://localhost:8080/



### Run in prod


```
npm start
```

navigate to http://localhost:3000/
